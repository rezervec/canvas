### Функционал
---
При нажатии на кнопку - падает рандомная картинка.

**Цель этого проекта:**
Сравнить производительность 3ёх схожих по задаче приложений, в каждом из которых основной функционал реализован на:
- Canvas
- Java Script (чистый)
- CSS

:black_small_square:

### Установка
---
Репозиторий содержит 3 HTML-файла. В названии каждого файла указан инструмент, который выполняет основной функционал приложения.
Для работы приложения необходим репозиторий ```./images``` и HTML-файл.
