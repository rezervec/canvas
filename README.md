### Функционал
---
При нажатии на кнопку - падает рандомная картинка.

**Цель этого проекта:**
Сравнить производительность 3ёх схожих по задаче приложений, в каждом из которых основной функционал реализован на:
- Canvas
- Java Script (чистый)
- CSS

:black_small_square:

### Установка
---
Репозиторий будет содержать 3 файла. В названии каждого HTML-файла после знака 'v' будет указан основной инструмент, который выполняет основной функционал приложения.
Для работы приложения необходим репозиторий ```./images``` и HTML-файл.
